# Folder Icon Changer by IDS

Aplicativo portátil para Windows que troca o ícone de pastas usando imagens comuns como PNG, JPG, WEBP, BMP e ICO.

Desenvolvido pela [IntelligentDevSolutions](https://intelligentdevsolutions.com/).

Contato do dev: **(11) 96637-3319**

## Download

Baixe a versão portátil em:

[FolderIconChanger-Windows-v1.0.zip](dist/FolderIconChanger-Windows-v1.0.zip)

## Como usar

1. Execute `Folder Icon Changer.exe`.
2. Selecione a pasta que deseja personalizar.
3. Selecione uma imagem.
4. Confira a prévia e clique em `Aplicar Ícone`.

Para desfazer, selecione a pasta e clique em `Restaurar Padrão`.
